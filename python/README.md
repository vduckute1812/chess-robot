Prefer the C++ apps documented in the repo root [`README.md`](../README.md) and [`camera_development.md`](../camera_development.md).

```bash
# From repo root, with the existing venv:
./env/bin/python -m src.main
```

Requires `PYTHONPATH` to include this folder or running with `cd python` and adjusting imports — easiest is:

```bash
cd python
../env/bin/python -m src.main
```
